# Ember Peak Pager

A simple, flexible and customizable input UI component for masking data.

## Compatibility

* Ember.js v3.16 or above
* Ember CLI v2.13 or above
* Node.js v10 or above

## Installation

```
ember install @summit-electric-supply/ember-peak-masked
```

## Usage

Example:

```
<PeakMasked />
```
